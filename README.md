# ClickFX

> 为鼠标点击添加酷炫的全局视觉特效

ClickFX 是一款轻量级 Windows 桌面工具，在你每次点击鼠标时，屏幕上会播放精心设计的动画效果

---

## ✨ 特性

- **即装即用** — 单个 `ClickFX.exe`，双击运行，无需安装，无外部依赖
- **对系统零干扰** — 不拦截鼠标操作，不抢焦点，不进任务栏
- **多显示器支持** — 自动为每个屏幕创建独立的动画覆盖层
- **左右键独立配色** — 左键和右键可以设置不同的动画颜色
- **自定义颜色** — 通过 HEX 色值或系统颜色选择器自由调色
- **配置持久化** — 设置自动保存到 `%APPDATA%\ClickFX\config.json`，重启不丢失
- **极低资源占用** — 动画仅在点击时触发，空闲时几乎无 CPU 开销

### 🎬 录屏 & 演示利器

做教程、录屏、投屏演示时，观众经常看不清你点了哪里。ClickFX 在每次点击处自动显示醒目的动画标记，**让观众一眼就能跟上你的操作节奏**，无需额外标注或后期加工。左键 / 右键使用不同颜色，操作意图更加清晰。

## 🚀 快速开始

### 运行要求

- Windows 10 / 11
- .NET Framework 4.8（Windows 10 1903+ 已内置）

### 使用方法

1. 下载 `ClickFX.exe`
2. 双击运行 — 托盘区会出现 ClickFX 图标
3. 现在点击鼠标，屏幕上就会出现特效动画！

### 退出

右键点击托盘区的 ClickFX 图标 → **退出 ClickFX**

---

## 🎨 设置

右键点击托盘区图标 → **设置**，打开配置窗口：

| 选项 | 说明 |
|------|------|
| **动效** | 从下拉框选择当前使用的动画效果 |
| **左键颜色** | 设置鼠标左键点击时的动画颜色（HEX 色值，如 `#508CFF`） |
| **右键颜色** | 设置鼠标右键点击时的动画颜色（HEX 色值，如 `#FF6060`） |
| **选择** | 打开系统颜色对话框，可视化取色 |
| **恢复默认** | 一键恢复左键 `#508CFF`、右键 `#FF6060` 的默认配色 |

设置完成后点击 **确定** 保存，或 **取消** 放弃更改。

## ❓ 常见问题

**Q: ClickFX 会影响鼠标操作吗？**
A: 不会。ClickFX 仅监听鼠标事件并通过 `CallNextHookEx` 原样透传，不会拦截、延迟或修改任何点击操作。

**Q: 会出现在任务栏吗？**
A: 不会。ClickFX 仅在系统托盘区显示图标，不会占用任务栏。

**Q: 如何彻底退出？**
A: 右键点击托盘图标 → 退出 ClickFX。

**Q: 配置文件在哪里？**
A: `%APPDATA%\ClickFX\config.json`（通常为 `C:\Users\你的用户名\AppData\Roaming\ClickFX\config.json`）。

**Q: 支持开机自启吗？**
A: 当前版本暂不支持，后续版本计划加入。
